# Challenge 01 - Highest Paid Employee in April 2012

## Problem Statement

Find all employees receiving the highest net pay during April 2012.

### Output Columns

- EMPID
- EMPNAME
- DEPTID
- DEPTNAME

### Tables

- EMP_INFO
- DEPT_INFO
- EMP_PAY_INFO

### Note

More than one employee can have the same highest salary.
